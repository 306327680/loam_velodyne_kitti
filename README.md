# loam_velodyne_kitti
 This package is a simple modified copy of [loam_velodyne git repository](https://github.com/laboshinl/loam_velodyne) from **laboshinl** to work with the KITTI dataset.
 
How to build:

```
$ mkdir build
$ cd build
$ make
```
Running:
```
$ ./main "KITTI Dataset Path" "NumberOfSequence"
